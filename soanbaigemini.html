Soạn bài Gemini - Tạo bài học hoàn chỉnh

Dán ảnh sách → Gemini phân tích + sinh nội dung + hình ảnh minh họa → Xuất Word + copy vào Editor

Chạy tốt trong Gemini Canvas
Luồng tích hợp (tất cả trong file này):
1. Dán/Ctrl+V nhiều ảnh screenshot sách vào vùng Ảnh đầu vào.
2. Nhấn "Tạo bài học đầy đủ từ ảnh" (gọi Gemini trực tiếp với key của bạn + ảnh).
3. Tool nhận kết quả, parse thành đúng cấu trúc hệ thống.
4. Tự động tạo danh sách prompt hình ảnh chuẩn sách giáo khoa.
5. Bạn gen ảnh với Gemini, dán ảnh vào tool nếu muốn preview.
6. Xuất Word hoàn chỉnh (có vị trí hình + prompt) hoặc copy blocks sang editor chính.

1. Đầu vào

Cần key Gemini (tạo tại aistudio.google.com). Chỉ dùng local, không gửi đâu cả.
Ctrl + V dán ảnh chụp màn hình vào đây
Chụp trang sách → dán trực tiếp (nhiều ảnh OK)
Chụp màn hình → Ctrl + V nhiều lần để thêm ảnh. Sau đó nhấn nút lớn bên phải để gọi Gemini trực tiếp.

2. Tạo nội dung

Tool sẽ gửi ảnh + prompt chi tiết đến Gemini API của bạn và nhận kết quả ngay trong file này.
Prompt đã tạo (copy cái này cho Gemini):
→ Dùng nút Copy ở trên. Đừng kéo chuột chọn chữ trong ô này để tránh nút Gemini nổi lên.

3. Kết quả từ Gemini (dán vào đây)

Mẹo: Dán xong nhấn "Phân tích" → dùng nút Copy của từng phần (không cần quét chọn trong ô này).
Chưa có nội dung.
1. Dán kết quả Gemini vào ô bên trái
2. Nhấn "Phân tích & Xem trước"
3. Xuất Word rồi copy từng khối vào đúng tab của hệ thống.

Hướng Dẫn Sử Dụng Nhanh

1. Cấu hình AI (Bắt buộc)

Để hệ thống hoạt động, bạn cần nạp API Key:

  • Bấm nút Cấu hình ở góc trên bên phải.
  • Gemini Keys: Nạp file .txt chứa danh sách API Key của Google (mỗi dòng 1 key).
  • Hugging Face Token: Nhập token (bắt đầu bằng hf_) để sử dụng model Qwen 2.5 miễn phí khi Gemini quá tải.

2. Các tính năng chính

  • Dạy học từ Google Slide: Tích hợp Google Slides vào hệ thống để giảng dạy trực tiếp không cần chuyển tab.
  • Dùng AI để vẽ hình học: Nhập đề bài, AI sẽ hỗ trợ vẽ hình.
  • Tạo Đề Ma Trận: Upload ảnh bảng ma trận, AI tự động đọc rồi sinh đề thi chuẩn.
  • Trộn Đề Thi: Upload file Word/PDF rồi xáo trộn câu hỏi thành nhiều mã đề.
  • Đề Kiểm Tra: Sinh đề từ ảnh SGK hoặc tạo đề tương tự từ PDF.
  • Thi Online: Tổ chức thi trắc nghiệm trên điện thoại, máy tính.

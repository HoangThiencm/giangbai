Cấu Hình Hệ Thống AI

Cần nạp 2 file để hệ thống hoạt động

Cần nạp 2 file để hệ thống hoạt động:

1. API Keys.txt - Mỗi dòng là 1 API Key

2. Modules.txt - Mỗi dòng là 1 module Gemini

File API Keys.txt

Mỗi dòng chứa 1 API Key Gemini

File Modules.txt

Mỗi dòng chứa 1 module (ví dụ: gemini-2.5-flash)

Hướng dẫn:

  • • Tạo file API Keys.txt với mỗi dòng là 1 API Key
  • • Tạo file Modules.txt với mỗi dòng là 1 module
  • • Hệ thống sẽ tự động ghi nhớ cấu hình
